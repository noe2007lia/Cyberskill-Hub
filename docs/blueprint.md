# **App Name**: Cyberskill Hub

## Core Features:

- User Authentication: Enable secure learner login and registration using Firebase Authentication.
- Homepage & Course Overview: Display a welcoming homepage with a course title, explanation, and a 'Start Training' button.
- Interactive Learning Modules: Present structured cybersecurity modules, each with explanation text, supporting visuals, an interactive task/question, and a 'Mark as complete' button.
- AI-Powered Concept Clarification: Provide a tool to generate on-demand, concise AI-powered explanations or summaries for complex cybersecurity concepts within modules.
- Progress Tracking & Gamification: Track and display learner progress through modules, including a visual progress bar or percentage. Award points for module completion and a badge for finishing all modules, saving data in Firestore.
- Kahoot Quiz Integration: Include a button at the end of the training linking directly to the specified external Kahoot quiz: https://create.kahoot.it/share/cybersecurity-schulung/fea73928-f7c6-4996-9471-51d1f30ca16f.
- Admin/Coach Dashboard: Provide a simple overview for coaches to view learner names, completed modules, and their overall progress percentage from Firestore.

## Style Guidelines:

- Primary color: Vibrant blue (#4690FC) for key interactive elements, symbolizing trust and digital technology, suitable for a dark theme.
- Background color: Dark, muted blue-gray (#272C34) as the main background, offering a professional and focused atmosphere. Text should primarily be white for readability.
- Accent color: Bright cyan (#84EAFF) for highlighting important information, calls-to-action, and adding visual contrast against the dark background.
- Headline font: 'Space Grotesk' (sans-serif) to convey a modern, tech-oriented, and scientific feel.
- Body text font: 'Inter' (sans-serif) for exceptional readability across all devices, crucial for learning content.
- Use clean, minimalist, and modern icons with cybersecurity themes (e.g., lock, shield, target) to visually support module content and gamification elements.
- Employ a clean, intuitive, and spacious layout to prevent complexity. The design should be fully responsive, ensuring optimal user experience on both laptop and mobile devices.
- Implement subtle, smooth animations for state changes, such as module completion, progress updates, and navigation, to enhance the professional and modern feel without distracting the user.
