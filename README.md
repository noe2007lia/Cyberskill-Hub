# Cyberskill Hub

Modern cybersecurity training platform for everyone.

## Overview
Cyberskill Hub is a modern, interactive learning platform designed to teach the fundamentals of cybersecurity. From identifying phishing emails to creating secure passwords, the hub provides structured modules with practical tasks and gamified progress tracking.

## Features
- **Structured Modules**: Practical lessons on Phishing, Passwords, Social Engineering, and more.
- **Gamification**: Earn XP and track your progress with a visual dashboard.
- **Persistence**: Your progress is automatically saved to your browser's local storage.
- **Coach Dashboard**: Administrative view for monitoring learner progress.
- **Interactive Quiz**: Integrated final assessment via Kahoot.

## Technical Stack
- **Framework**: Next.js (App Router)
- **UI Components**: React, Shadcn UI, Radix UI
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Fonts**: Space Grotesk (Headline), Inter (Body)
